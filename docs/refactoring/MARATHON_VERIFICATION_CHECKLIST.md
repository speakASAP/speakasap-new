## 7.2 Redirect chain

From browser or `curl`:

1. **speakasap.com/marathon** → redirects to **marathon.alfares.cz/** (or configured MARATHON_URL).
2. **speakasap.com/marathon/german/** (or `/marathon/de/`) → redirects to **marathon.alfares.cz/** with path preserved (e.g. `/de/` if portal sends path).
3. Final URL shows the correct SPA view (home or landing).

*Note:* Portal redirect may send full marathon.alfares.cz URL; path preservation depends on portal redirect implementation.
